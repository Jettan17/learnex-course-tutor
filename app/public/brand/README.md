# LearnEx Brand Assets

Official logo and brand assets for the LearnEx learning platform.

## Logo Variants

| File | Preview | Use Case | Dimensions |
|------|---------|----------|------------|
| `logo.svg` | Icon only | App icon, small spaces, favicons | 48x48 |
| `logo-full.svg` | Icon + text | **Portfolio, headers, social media** | 200x48 |
| `logo-full-dark.svg` | Icon + text (light) | Dark backgrounds | 200x48 |
| `logo-stacked.svg` | Icon above text | Square formats, app stores | 120x120 |
| `favicon.svg` | Compact icon | Browser favicon | 32x32 |

## Quick Access URLs

When the app is deployed, access logos directly at:

```
https://your-domain.com/brand/logo.svg
https://your-domain.com/brand/logo-full.svg
https://your-domain.com/brand/logo-full-dark.svg
https://your-domain.com/brand/logo-stacked.svg
https://your-domain.com/brand/favicon.svg
```

## Typography

The wordmark uses the system font stack:
```css
font-family: system-ui, -apple-system, 'Segoe UI', Roboto, sans-serif;
font-weight: 700;
letter-spacing: -0.5px;
```

## Usage Guidelines

### Do
- Use the full logo (`logo-full.svg`) for portfolio and marketing
- Maintain the aspect ratio when scaling
- Use `logo-full-dark.svg` on dark backgrounds
- Provide adequate spacing around the logo

### Don't
- Modify the colors or proportions
- Add effects like shadows or gradients
- Place on busy backgrounds without contrast
- Stretch or distort the logo

## Download

Right-click any logo file and "Save As", or access them directly from the `/brand/` directory.
